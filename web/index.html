Welcome to TigerVNC

TigerVNC is a suite of VNC servers and clients that have a focus on performance and remote display functionality. Originally this software was based on the (never released) VNC 4 branch of TightVNC.

Downloads

The project has not yet made any official releases. If you want to take the software for a spin then check the subversion pages.

Community

The project has three mailing lists set up:

tigervnc-announce
Moderated announcement list for things such as a new release.
tigervnc-users
Support and general discussion list for users of TigerVNC. Please use this list for bug reports.
tigervnc-devel
Discussion list for TigerVNC developers. If you have a patch or some development ideas then please send them here.

There is also an IRC channel #tigervnc on Freenode. This channel is open to users and developers alike. Please note that not all developers are present in this channel, so use the mailing lists if it is something critical.
