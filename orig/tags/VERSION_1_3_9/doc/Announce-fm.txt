This release is a long-awaited new stable version. Changes in the Unix version include support for the latest RFB protocol version 3.8, VNC4-compatible parsing of host:display strings, and a number of bug fixes and portability fixes. Also, there was a lot of changes in the built-in Java viewer, such as scaling, automatic encoding selection, support for ZRLE encoding, and a workaround for the Tab key problem.
